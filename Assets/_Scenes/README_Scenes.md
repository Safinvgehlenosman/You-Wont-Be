# Scene Naming Convention

## Bootstrap scenes
- Prefix: `BS_`
- Example: `BS_Main`

## Gameplay scenes
- Prefix: `LV_`
- Examples: `LV_DevGym`, `LV_Village01`

## Dev/test scenes
- Place in `Assets/_Scenes/Dev/` or name with `LV_Dev*`

## Third-party/vendor scenes
- Remain in `Assets/Scenes/` (do not move or rename)
